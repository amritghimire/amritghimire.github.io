## Getting Started

### First Launch

1. **Create Your First Profile**
   - When you first open Biblioform, you'll be prompted to create a profile
   - Profiles let you manage multiple libraries (e.g., personal, family, office)
   - You can create additional profiles anytime from the main screen

2. **Add Your First Book**
   - Tap the **+** button in the Library tab
   - Choose from:
     - **Search**: Enter title, author, or ISBN
     - **Scan**: Use your camera to scan a barcode
     - **Bulk Scan**: Scan multiple books quickly
     - **Manual**: Enter book details manually

3. **Explore the Tabs**
   - **Library**: View all your books
   - **Categories**: Browse by genre
   - **Locations**: Organize by physical location
   - **Stats**: Track your reading goals

## How-To Guides

### How to Add Books

**Method 1: Barcode Scanning**
1. Tap the **+** button in Library tab
2. Tap **"Scan"** or **"Bulk"**
3. Point camera at book's barcode
4. Book is automatically added with metadata

**Method 2: Search**
1. Tap the **+** button
2. Enter book title, author, or ISBN
3. Tap **Search**
4. Select book from results
5. Tap to add

**Method 3: Manual Entry**
1. Tap the **+** button
2. Tap **"Manual"**
3. Fill in book details
4. Tap **"Add"**

### How to Organize by Location

**Create a Location**
1. Go to **Locations** tab
2. Tap **+** button
3. Enter location name
4. Choose location type (Shelf, Room, etc.)
5. Select color and icon
6. Optionally set as sub-location of another location

**Enable Grid Layout**
1. Create or edit a location
2. Toggle **"Has Grid"** on
3. Set number of rows and columns
4. Save location

**Assign Books to Location**
1. Open a book's detail view
2. Tap **"Set Location"** or **"Change"**
3. Select location from list
4. If location has grid, select grid position
5. Book is now assigned

**Bulk Assign to Location**
1. In Library view, tap selection icon (checkmark)
2. Select multiple books
3. Tap menu → **"Assign to Location"**
4. Choose location and optional grid position
5. All selected books are assigned

### How to Use Categories

**Assign Category to Book**
1. Open book detail view
2. Tap **"Set Category"** or **"Change"**
3. Select category from list
4. Category is assigned

**Bulk Assign Categories**
1. In Library or Categories view, enter selection mode
2. Select multiple books
3. Tap **"Assign Category"**
4. Choose category
5. All selected books get the category

**Browse by Category**
1. Go to **Categories** tab
2. See all books organized by genre
3. Tap category to expand/collapse
4. Tap book to view details

### How to Track Reading Progress

**Update Reading Status**
1. Open book detail view
2. Tap reading status (Unread, Reading, Read, etc.)
3. Status updates automatically

**Track Page Progress**
1. Open book detail view
2. Set current page number
3. Progress percentage updates automatically
4. View estimated days to finish

**Mark as Read**
1. Set status to **"Read"**
2. Optionally set finish date
3. Add rating (1-5 stars)
4. Add notes about the book

### How to Set Reading Goals

1. Go to **Stats** tab
2. Tap **"Set Goal"** or edit existing goal
3. Enter target number of books
4. Enter target number of pages
5. Goal appears on stats dashboard
6. Progress updates automatically as you read

### How to Use Bulk Operations

**Bulk Category Assignment**
1. Enter selection mode (checkmark icon)
2. Select multiple books
3. Tap **"Assign Category"**
4. Choose category
5. All selected books are updated

**Bulk Location Assignment**
1. Select multiple books in Library view
2. Tap menu → **"Assign to Location"**
3. Choose location
4. Optionally select grid position
5. All books are assigned

**Bulk Scan Books**
1. Tap **+** button → **"Bulk"**
2. Point camera at barcodes
3. Books are added automatically
4. Pause/resume scanning as needed
5. View list of scanned books

## Troubleshooting

### Books Not Syncing

**Problem**: Books added on one device don't appear on another

**Solutions**:
1. Check iCloud is enabled in Settings
2. Ensure you're signed into the same iCloud account on all devices
3. Wait a few moments for sync to complete
4. Force quit and reopen the app
5. Check internet connection

### Barcode Not Scanning

**Problem**: Camera doesn't recognize barcode

**Solutions**:
1. Ensure good lighting
2. Hold camera steady
3. Move closer or farther from barcode
4. Clean camera lens
5. Try manual entry if barcode is damaged

### Book Not Found in Search

**Problem**: Book doesn't appear in search results

**Solutions**:
1. Try searching by ISBN instead
2. Search by author name
3. Use manual entry
4. Check spelling of title/author
5. Book may not be in OpenLibrary database

### Location Grid Not Showing

**Problem**: Grid layout doesn't appear for location

**Solutions**:
1. Ensure "Has Grid" is enabled for location
2. Set number of rows and columns
3. Save location settings
4. Refresh location detail view

### App Crashes

**Problem**: App closes unexpectedly

**Solutions**:
1. Force quit and reopen app
2. Restart your device
3. Update to latest app version
4. Check available storage space
5. Contact support if issue persists

### Data Not Appearing

**Problem**: Books or locations disappear

**Solutions**:
1. Check iCloud sync status
2. Ensure you're viewing correct profile
3. Check filters aren't hiding items
4. Verify data in iCloud settings
5. Contact support for data recovery

## Frequently Asked Questions

### General Questions

**Q: Is Biblioform free?**
A: Yes, Biblioform is free to download and use with no in-app purchases.

**Q: Do I need an account?**
A: No account required! The app uses your Apple ID for iCloud sync, but no separate account is needed.

**Q: What devices does it work on?**
A: Biblioform works on iPhone, iPad, and Mac with iCloud sync across all devices.

**Q: Can I use it offline?**
A: Yes! You can use all features offline. iCloud sync happens when you're connected to the internet.

**Q: How many books can I add?**
A: There's no limit! Biblioform can handle collections of any size.

### Data & Privacy

**Q: Where is my data stored?**
A: Your data is stored locally on your device and in your private iCloud account. We cannot access your data.

**Q: Is my data private?**
A: Yes! All your data is private to you. We don't collect analytics, track usage, or share data with anyone.

**Q: Can I export my data?**
A: Your data is stored in iCloud, which you can access through iCloud settings. We're working on additional export options.

**Q: What happens if I delete the app?**
A: Your data remains in iCloud. Reinstalling the app will restore your library if iCloud sync is enabled.

### Features

**Q: Can I track e-books and audiobooks?**
A: Yes! You can set the format (Physical, E-book, Audiobook, PDF) for each book.

**Q: How do I track books I've lent to friends?**
A: Create a location with type "Person" and assign the book to that location. Set an expected return date.

**Q: Can I have multiple libraries?**
A: Yes! Create multiple profiles for different libraries (personal, family, office, etc.).

**Q: Does it work with Goodreads or other services?**
A: Currently, Biblioform is standalone. We're considering integration options for future updates.

**Q: Can I share my library with others?**
A: Not currently, but you can show your library on your device. Sharing features may be added in future updates.

### Technical

**Q: Why does the app need camera access?**
A: Camera access is only used for barcode scanning to add books quickly. No photos are stored.

**Q: Does it use internet?**
A: Internet is used for:
- Searching book metadata from OpenLibrary
- Downloading cover art
- iCloud sync

All other features work offline.

**Q: Will it work on older devices?**
A: Biblioform requires iOS 17+ / macOS 14+ or later. Check your device compatibility.

**Q: How do I update the app?**
A: Updates come through the App Store. Enable automatic updates or update manually.
